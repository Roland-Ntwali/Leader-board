# 📖 Leader board  <a name="Leader board"></a>


**Leader board** This is a simple website that displays a list of names and scores of different players.


## 🛠 Built With <a name="built-with"></a>

- HTML
- CSS
- JavaScript
- Webpack
  
### Tech Stack <a name="tech-stack"></a>

For this project design I am using form to create localstorage data and class object array to update the page.


### Key Features <a name="key-features"></a>

> **[CRUD Leader board ]**
  - Create Read, Update & Delete leader board item.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running follow these simple example steps.

### Prerequisites
- Computer
- Code editor (Vs code)

### Setup

- clone 'https://github.com/Roland-Ntwali/Leader-board.git'
- Setup Linters by running following commands in your terminal:

- npm install --save-dev stylelint@13.x stylelint-scss@3.x - - - stylelint-config-standard@21.x stylelint-csstree-validator@1.x (Install the StyleHint linters)

- npm install --save-dev eslint@7.x eslint-config-airbnb-base@14.x eslint-plugin-import@2.x babel-eslint@10.x

- npx stylelint "*/.{css,scss}" (checks for the CSS related errors in your project)

- npx stylelint "*/.{css,scss}" --fix (This will fix the linters error, if any)

- npm install --save-dev hint@7.x (Install the WebHint Linters)

- Run npx eslint . on the root of your directory of your project.

- npx hint . (You should see Finishing... on your terminal)

- set well your webpack
